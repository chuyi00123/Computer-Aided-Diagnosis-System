## 1.项目简介
本项目以以图像分割为中心，利用人工智能实现肾脏、肿瘤及囊肿的精准分割，并通过Flask框架将肾脏及其病灶分割模型部署到网络应用程序。前端使用简单的JavaScript，HTML和CSS，后端使用Python语言开发，框架基于Flask和Pytorch, 相关代码持续更新中......
